Assignment #4 "Elementary Compiler"
this project is submitted to the "01076262 Compiler Construction" subject, CE KMITL 2016
## Getting Started

- Create a compiler program

```shell
  $ flex TGS.l
  $ yacc -d TGS.y
  $ gcc lex.yy.c y.tab.c -o TGSProg
  $ ./TGSProg
```
